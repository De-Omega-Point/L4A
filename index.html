Most Common Actions

Add a customer

Saves a new person and gives them a QR token.

Use it when someone joins the loyalty program.

Scan a customer

Finds the customer from their QR code.

Use it at the counter when they show their card.

Use Revisit mode

Keeps the scanner open for walk-in visit points.

Use it when customers scan themselves as they arrive.

Redeem a reward

Marks the reward as given and saves the history.

Use it when Reward Ready appears.

Export customer list

Saves a CSV spreadsheet named mydata.

Use it when the owner wants a customer list.

Save backup

Saves customers, points, notes, and settings.

Use it before changing device or browser.

Restore backup

Loads a saved Loyalty-4-All backup file.

Use it when moving data back onto this device.

Send SMS/email

Opens the phone SMS app or email app.

Use it only when the customer has agreed.

Connect a service

Automatic campaigns need an SMS or email service.

Use it later when a secure service is ready.

Troubleshooting

Camera will not open

Allow camera permission, then try Scan QR again.

Use Find customer if the camera is blocked.

QR code will not scan

Move closer, add light, and keep the QR flat.

Print a larger QR if it keeps failing.

Customer cannot be found

Search by name, phone, email, or token.

Add the customer again only if they are not listed.

SMS/email does not open

Check the device has a SMS or email app set up.

Copy the message text if the app will not open.

Data looks missing

Check you are using the same browser and device.

Restore the latest backup if needed.

Backup restore not working

Choose a Loyalty-4-All JSON backup file.

If unsure, keep the old file and ask support.

Printed QR is blurry

Use PNG export or print a larger size.

Avoid tiny labels for customer cards.
